The <table> tag creates table. the <tr> tag makes table row. the <td> makes tables. the <th> tag makes tables with heading. the <link> tag links CSS, favicon and more.. (learn linking css in next course). to link favicon, here's code: <link rel="shortcut icon" type="image/x-icon" href="src"> or type link:favicon in VScode for shortcut.

The semantic tags make web easier to code! here's some of them: <header>, <main>, <aside>, <article> <footer> and more.. the <div> tag makes box model. its very important for CSS. the <form> tag makes forms. the <input> tag makes form inputs. the <label> tag makes form label. the <audio> tag imports audio. the <video> tag imports videos.
