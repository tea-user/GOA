The <p> tag allows you to create text. The <h1> to <h6> tags allow you to create heading texts. the <button> tag creates button and the <img> tag creates an image. The <br> tag breaks line and the <hr> tag makes line. you can change color of line by using attributes. attributes are placed in opening tag. (for ex: <hr color="green">. This will show green line).

The <a> tag generates a link. The <ol> tag generates ordered list. The <ul> tag generates unordered list. the <li> tag generates list item. the <center> tag makes things aligned centre (not recommended). the align attribute displays item on left/right or center.
